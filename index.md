# GuardianTimer Dokumentation

Willkommen auf der Dokumentationsseite für **GuardianTimer**, die iOS-App mit Totmannschalter.

## Projektübersicht

GuardianTimer ist eine App, die die Sicherheit von Alleinarbeitern oder Personen in Gefahrensituationen gewährleistet, indem sie regelmäßig Lebenszeichen (Standort und Zeit) an einen Empfänger sendet. Bleibt die Aktualisierung aus, wird ein Alarm ausgelöst.
