# GuardianTimer Docs

Dieses Repository beinhaltet die Dokumentation für die **GuardianTimer** App als Webseite.

## Veröffentlichung

Diese Dokumentation ist für **GitHub Pages** vorbereitet. 
Nach dem Push auf GitHub muss in den Repository-Einstellungen unter "Pages" der Branch (meist `main` oder `master`) als Source ausgewählt werden.

## Struktur

* `index.md`: Startseite der Dokumentation
* `_config.yml`: Konfiguration für das Jekyll-Theme (Minimal)
* `assets/`: Speicherort für Bilder und CSS-Dateien
